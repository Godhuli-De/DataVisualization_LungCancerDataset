# DataVisualization_LungCancerDataset
Using Pandas, Matplotlib, Seaborn to perform EDA on Lung Cancer Dataset

Plots included:
countplot
Bar plot
boxplot
Pie chart 
Violin plot
Stacked bar plot
Boxen plot
Swarm plot
Joint plot
Heatmap of Features
Heatmap of missing values
feature importances
